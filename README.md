# goit-algo2-hw-04

Домашнє завдання з курсу «Проєктування, конструювання та аналіз алгоритмів»
(Алгоритми та структури даних 2).

## Структура репозиторію

```
goit-algo2-hw-04/
├── task_1/                # Максимальний потік для логістики (Едмондс–Карп)
│   ├── main.py
│   └── README.md          # звіт із розрахунками та аналізом
└── task_2/                # Розширення префіксного дерева (Trie)
    ├── trie.py            # базовий клас Trie
    ├── main.py            # клас Homework з новими методами
    └── README.md
```

## Завдання 1 — Максимальний потік ([деталі](task_1/README.md))

Моделювання мережі логістики «термінали → склади → магазини» (20 вершин) та
пошук максимального потоку алгоритмом **Едмондса–Карпа**.

```bash
python task_1/main.py
```

**Результат:** максимальний потік = **115 одиниць**, що дорівнює мінімальному
розрізу (сумарна вихідна ємність терміналів). Звіт містить покроковий розрахунок,
підсумкову таблицю «Термінал → Магазин» та відповіді на аналітичні запитання.

## Завдання 2 — Trie: суфікси та префікси ([деталі](task_2/README.md))

Клас `Homework(Trie)` з методами `count_words_with_suffix(pattern)` та
`has_prefix(prefix)`, з валідацією вхідних даних і урахуванням регістру.

```bash
python task_2/main.py
```

## Вимоги

* Python 3.8+
* Зовнішніх залежностей немає (алгоритм Едмондса–Карпа реалізовано вручну).
